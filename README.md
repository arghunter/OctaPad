# OctoPad
An irregular octagon macropad. Why? I felt like it. 
## Features
- 1x OLED Screen
- 2x Rotary Encoders
- 4x MX Switches
- 16x Neopixel for Underglow

## BOM
- Seeed XIAO RP2040
- 4x MX-Style switches
- 2x EC11 Rotary encoders
- 1x 0.91 inch OLED displays
- 4x Blank DSA keycaps (White)
- 16x SK6812 MINI-E LEDs
